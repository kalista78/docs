## Screenshot Guidelines

### Dimensions
- **Recommended width:** 600-800px
- **Format:** PNG (preferred) or WebP
- **Compression:** Optimize for web

### Capture Tips
1. Use a clean Telegram interface
2. Hide personal information (use test accounts)
3. Show realistic but not real data
4. Ensure text is readable
5. Capture full context (include relevant buttons)

### Naming Convention
```
{section}-{description}.png
```

Examples:
- `buy-confirm.png`
- `bridge-quote.png`
- `rakeback-dashboard.png`

### Image Location
Place all images in `/acebotdocs/images/` directory.

Reference in MDX files:
```mdx
![Description](/images/filename.png)
```
