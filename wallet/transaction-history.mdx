### Transaction Hash

### Block

### From / To
- **From** â€” Your wallet address (for outgoing)
- **To** â€” The contract or address interacted with

### Value
Amount of native tokens transferred (if any).

### Gas Used
Amount of gas consumed by the transaction.

### Transaction Fee

## Reading Swap Transactions

For buy/sell transactions, look for:

### Input
What you sent (e.g., "10 S" or "1000 TOKEN").

### Output
What you received (shown in internal transactions or token transfers).

### Token Transfers
Section showing all token movements in the transaction.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Transaction not showing" icon="eye-slash">
    - Wait a few minutes for indexing
    - Refresh the explorer page
    - Verify you're on the correct chain's explorer
    - Check that the transaction actually confirmed
  </Accordion>

  <Accordion title="Shows wrong token amounts" icon="question">
    - Some explorers have delayed token detection
    - Try a different explorer for the same chain
    - Check the token transfers tab specifically
  </Accordion>

  <Accordion title="Can't find old transactions" icon="magnifying-glass">
    - Use the explorer's advanced search
    - Filter by date range if available
    - Search by the specific transaction hash if you have it
  </Accordion>
</AccordionGroup>

## Exporting History

For tax reporting or record keeping:

<Steps>
  <Step title="Visit Explorer">
    Go to your chain's block explorer.
  </Step>

  <Step title="Search Your Address">
    Enter your wallet address.
  </Step>

  <Step title="Export">
    Most explorers offer CSV export options:
    - Look for "Download CSV" or "Export"
    - Select date range if needed
    - Download transactions list
  </Step>
</Steps>

<Info>
For multi-chain activity, you'll need to export from each chain's explorer separately.
</Info>
