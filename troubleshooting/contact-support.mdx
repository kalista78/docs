# Contact Support

If you can't resolve an issue yourself, here's how to get help.

## Before Contacting Support

Try these first:

<Steps>
  <Step title="Check Documentation">
    Search this documentation for your issue. Most problems are covered in [Common Issues](/troubleshooting/common-issues).
  </Step>

  <Step title="Try Basic Fixes">
    - Send `/start` to reset the bot
    - Use `/balance` to refresh balances
    - Check you're on the correct chain
    - Verify your wallet has gas
  </Step>

  <Step title="Search Community">
    Your question may have been answered in community channels.
  </Step>
</Steps>

## What to Include in Support Requests

Help us help you faster by including:

### For Transaction Issues
- Transaction hash (if available)
- Token address involved
- Chain you were trading on
- Amount you were trying to trade
- Error message (screenshot if possible)
- Time the issue occurred

### For Wallet Issues
- Wallet address (public address, NOT private key)
- Chain involved
- What you were trying to do
- Any error messages

### For Bot Issues
- What action triggered the problem
- Any error messages shown
- Your Telegram username (for account lookup)

## Example Support Request

**Good request:**
```
Issue: Trade failed with slippage error

Details:
- Token: 0x1234...5678 on Sonic
- Amount: 50 S buy
- Slippage: 15%
- Error: "INSUFFICIENT_OUTPUT_AMOUNT"
- Time: Today at 3:45 PM UTC
- TX hash: 0xabcd...

I've tried increasing slippage to 30% but still failing.
```

**Poor request:**
```
Trade not working please help
```

The more details you provide, the faster we can help.

## Security Warning

<Warning>
**AceBot support will NEVER:**
- DM you first
- Ask for your private key
- Ask for your seed phrase
- Ask for wallet passwords
- Request remote access to your device
- Ask you to send tokens to "verify" your wallet

**Anyone doing this is a SCAMMER.**
</Warning>

## Reporting Scammers

If someone tries to scam you:

<Steps>
  <Step title="Don't Engage">
    Stop all communication immediately.
  </Step>

  <Step title="Block Them">
    Block the scammer on Telegram.
  </Step>

  <Step title="Report to Telegram">
    Use Telegram's report function.
  </Step>

  <Step title="Warn Community">
    Post in the community chat (without clicking any links they sent).
  </Step>
</Steps>

## Response Times

| Channel | Typical Response |
|---------|-----------------|
| Community Chat | Minutes to hours (community help) |
| Official Support | 24-48 hours |
| Critical Issues | Prioritized faster |

Complex issues may require investigation and take longer.

## What Support Can Help With

**We can help with:**
- Transaction troubleshooting
- Account issues
- Feature questions
- Bug reports
- General guidance

**We cannot help with:**
- Private key recovery (we don't have access)
- Reversing blockchain transactions
- Recovering funds sent to wrong addresses
- Issues with external services/tokens

## Feature Requests and Feedback

We actively read community feedback and prioritize based on demand.
