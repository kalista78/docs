# Create Your Wallet

AceBot creates your first wallet automatically when you start the bot. You can also create additional wallets anytime.

## Automatic Wallet Creation

When you first open AceBot and tap **Start**, the bot:

1. Creates a new wallet with a unique address
2. Encrypts your private key with AES-256-GCM
3. Assigns it a default name ("Wallet 001")
4. Sets it as your default trading wallet

<Info>
Your private keys are encrypted and stored securely. They are never displayed in Telegram chat.
</Info>

## Import an Existing Wallet

If you have an existing wallet, you can import it using your private key.

<Steps>
  <Step title="Open Wallet Manager">
    Tap **ðŸ‘› Wallets** from the main menu.
  </Step>

  <Step title="Select Import Wallet">
    Tap the **Import Wallet** button.
  </Step>

  <Step title="Enter Private Key">
    Paste your private key (64-character hex string with optional 0x prefix).

    <Warning>
    Your message containing the private key is **automatically deleted** for security. The key is encrypted before storage.
    </Warning>
  </Step>

  <Step title="Name and Confirm">
    Enter a name for the imported wallet. It's now ready to use.
  </Step>
</Steps>

## Set Your Default Wallet

Your default wallet is automatically selected for all transactions.

<Steps>
  <Step title="Open Wallet Manager">
    Tap **ðŸ‘› Wallets** from the main menu.
  </Step>

  <Step title="Select a Wallet">
    Tap on the wallet you want to set as default.
  </Step>

  <Step title="Set as Default">
    Tap **Set as Default**. A checkmark appears next to your default wallet.
  </Step>
</Steps>

## Wallet Display

In the wallet list, each wallet shows:
- **Wallet name** with chain emoji
- **Balance** in the native token
- **Checkmark** if it's your default wallet
- **Last 4 characters** of the address for quick reference
