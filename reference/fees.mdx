# Fee Structure

Transparent breakdown of all fees you'll encounter when using AceBot.

## Trading Fees

### Platform Fee
| Fee | Rate |
|-----|------|
| **Trading Fee** | 1% of trade volume |

This fee applies to all buy and sell transactions.

### How Platform Fee Works
When you trade:
1. You pay 1% of the trade volume as platform fee
2. Based on your rakeback tier, you earn back 0-30%
3. If you were referred, you earn an additional 10% back

**Net fee examples:**

| Your Tier | Referral Bonus | Effective Fee |
|-----------|----------------|---------------|

| Bronze | No | 0.95% |
| Silver | No | 0.90% |
| Gold | No | 0.85% |
| Platinum | No | 0.80% |
| Diamond | No | 0.75% |
| Apex | No | 0.70% |
| Any tier | +10% | -0.10% additional |

**Maximum rakeback:** 40% (minimum effective fee: 0.60%)

## Bridge Fees

When using the cross-chain bridge:

| Fee Type | Description |
|----------|-------------|
| **Bridge Protocol Fee** | Set by the bridge protocol (Li.Fi) |
| **Source Chain Gas** | Gas to initiate the bridge |
| **Destination Gas** | Sometimes included, sometimes paid separately |

Bridge fees are shown before you confirm each transaction.

**Typical total bridge cost:** $0.50 - $5.00 depending on route and congestion.

## Wallet Operation Fees

### Fund Distribution / Consolidation
| Fee | Rate |
|-----|------|
| **Platform Fee** | 0.1% of transferred amount |
| **Gas Fees** | Per transaction (network fee) |

These fees apply when distributing funds to multiple wallets or consolidating from multiple wallets.

### Regular Withdrawals
| Fee | Rate |
|-----|------|
| **Platform Fee** | None |
| **Gas Fee** | Standard network fee |

Standard withdrawals to external addresses only cost gas.

## Wrap/Unwrap Fees

| Fee | Rate |
|-----|------|
| **Platform Fee** | None |
| **Gas Fee** | Standard network fee |

Converting between native and wrapped tokens only costs gas.

## Limit Order / DCA Fees

When your automated orders execute:

| Fee Type | Rate |
|----------|------|
| **Trading Fee** | 1% (same as manual trades) |
| **Gas Fee** | Standard network fee per execution |

For DCA orders, each individual transaction incurs:
- The trading fee (with rakeback applied)
- A gas fee

**DCA cost consideration:** Many small DCA transactions can add up in gas. Consider larger, less frequent intervals if gas costs are significant.
